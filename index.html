Get stuff done

Getsdone is an opinionated task management app that makes adding and tracking tasks simple. Collaborate with colleagues, friends, family, gain valuable insight into your efficiency. Download the Getsdone app free today!



Easy to Use

Getsdone's interface is similar to popular social media apps, create tasks as easily as sending tweets. @someone to assign tasks, create hashtags, set priorities.

Insightful

Analyze your ability to get things done in real-time, see the last 5 day's progress, average time to close, gain levels as you complete tasks.

Collaborative

Assign tasks to others, see their progress, get notified, and get things done as a team.
